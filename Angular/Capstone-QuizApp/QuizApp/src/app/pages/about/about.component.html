Advanced Quiz Application - Documentation

Project Overview

The Advanced Quiz Application is a full-fledged Angular application that allows users to take quizzes, view results, and access a leaderboard. Admin users can create quizzes.

Key Features

  • User authentication (login/logout)
  • Role-based access (admin & users)
  • Quiz creation (admin-only)
  • Quiz taking functionality
  • Leaderboard tracking
  • JSON Server as a backend
  • Angular Router for navigation
  • RxJS for state management & async operations

Authentication

Authentication is handled using the AuthService. User data is stored in localStorage for persistence.

Routing

The application uses Angular Router with guards to protect quiz access and ensure only admins can create quizzes.

Quiz Management

Users can take quizzes, and once completed, responses are stored in JSON Server to prevent retaking the same quiz.

Leaderboard

The leaderboard ranks users based on their quiz performance. We have global leaderboard as well as leaderboard for each category
